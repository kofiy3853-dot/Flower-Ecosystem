# Plan: Multi-Step Course Creation Wizard

## Goal
Create a professional 7-step course creation wizard (like Teachable/Udemy) for instructors. Replace the existing minimal modal with a dedicated full-page wizard.

## Scope
- New `create-course.html` page with 7-step wizard UI
- Client-side state management (single JSON object)
- Expanded API endpoints to accept all course fields
- New DB tables for curriculum (sections + lessons)
- File upload integration (thumbnail, promo video, resources)
- Link from instructor dashboard

## Files to Create

### 1. `create-course.html` — Main wizard page

Full-page layout with:
- **Left sidebar**: Step list (7 items) with progress indicators
- **Center content**: Active step form
- **Right sidebar** (desktop): Course summary (lesson count, video count, resources, duration)
- **Bottom bar**: Back / Save Draft / Next buttons

**7 Steps:**

| Step | Title | Fields |
|------|-------|--------|
| 1 | Course Details | Title, subtitle, category (select), level (select), language, short description (200 char), full description (textarea), learning outcomes (dynamic list), requirements (dynamic list), target audience (checkboxes) |
| 2 | Curriculum | Sections with nested lessons. Each lesson: title, type (video/article/pdf/download/quiz/assignment), content/url. Drag handle for reorder. Add lesson / Add section buttons. |
| 3 | Media | Thumbnail upload (drop-zone, 1600x900 recommended), promo video upload, gallery images |
| 4 | Resources | File upload for PDFs, worksheets, templates. Name + file per resource. |
| 5 | Assessments | Quiz builder (title, passing score, questions with options), assignment builder (title, instructions, deadline), certificate toggle |
| 6 | Pricing | Free/Paid radio, price input, discount price, enrollment limit, visibility (public/private/invite-only) |
| 7 | Publish | Checklist (thumbnail, description, lessons, quiz, resources, preview), Save Draft / Preview / Submit for Review buttons |

**Client-side state:**
```js
const courseData = {
    // Step 1
    title: '', subtitle: '', category: '', level: 'Beginner', language: 'English',
    short_description: '', description: '',
    learning_outcomes: [], requirements: [], target_audience: [],
    // Step 2
    sections: [{ id, title, lessons: [{ id, title, type, content, video_url, duration }] }],
    // Step 3
    thumbnail_url: '', promo_video_url: '', gallery: [],
    // Step 4
    resources: [{ id, name, file_url, file_type }],
    // Step 5
    quiz: { title, passing_score, questions: [{ question, options, correct_answer }] },
    assignment: { title, instructions, deadline },
    has_certificate: true,
    // Step 6
    is_free: true, price: 0, discount_price: 0, enrollment_limit: 0, visibility: 'public',
    // Step 7 (computed)
    status: 'draft'
};
```

**Key UI patterns:**
- Each step validates before allowing "Next"
- Progress bar at top shows current step
- Left sidebar highlights active step, shows checkmark on completed steps
- Auto-save to localStorage every 30 seconds
- Resume from localStorage if page is reloaded

### 2. `js/create-course.js` — Wizard logic

Separate JS file (not inline) containing:
- Step navigation (next/prev/jump)
- Form validation per step
- Dynamic list management (add/remove outcomes, requirements, lessons)
- Curriculum builder (add section, add lesson, reorder, delete)
- File upload handlers using existing `image-upload.js` widget
- API submission (POST to create, PUT to update)
- localStorage save/restore
- Course preview generation

### 3. `styles/create-course.css` — Wizard-specific styles

Layout styles for:
- 3-column layout (step sidebar | content | summary sidebar)
- Step indicator with active/completed states
- Curriculum builder (nested drag-and-drop list)
- Form field styles (consistent with existing patterns)
- Mobile responsive (stack to single column, bottom nav buttons)

## Files to Modify

### 4. `routes/learning.js` — Expand course endpoints

**POST /api/courses** — Accept all fields:
```js
const { title, subtitle, description, short_description, category, level, language,
        learning_outcomes, requirements, target_audience,
        thumbnail_url, promo_video_url, gallery,
        is_free, price, discount_price, enrollment_limit, visibility,
        has_certificate, is_featured, tags } = req.body;
```
Insert with all columns. Set `is_published = false` (draft).

**PUT /api/courses/:id** — Accept same fields + `is_published`.

**New: POST /api/courses/:id/curriculum** — Save sections and lessons:
- Accept `{ sections: [{ title, lessons: [{ title, type, content, video_url, duration }] }] }`
- Delete existing sections/lessons for this course, re-insert (full replace)
- Update `lesson_count` on courses table

**New: POST /api/courses/:id/resources** — Save downloadable resources

**New: POST /api/courses/:id/quiz** — Save quiz with questions

**New: POST /api/courses/:id/assignment** — Save assignment

### 5. `sql/schema.sql` — New tables

```sql
CREATE TABLE IF NOT EXISTS learning.course_sections (
    id UUID PRIMARY KEY DEFAULT uuid_generate_v4(),
    course_id UUID NOT NULL REFERENCES learning.courses(id) ON DELETE CASCADE,
    title VARCHAR(255) NOT NULL,
    sort_order INT DEFAULT 0,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);

CREATE TABLE IF NOT EXISTS learning.course_lessons (
    id UUID PRIMARY KEY DEFAULT uuid_generate_v4(),
    section_id UUID NOT NULL REFERENCES learning.course_sections(id) ON DELETE CASCADE,
    course_id UUID NOT NULL REFERENCES learning.courses(id) ON DELETE CASCADE,
    title VARCHAR(255) NOT NULL,
    type VARCHAR(50) DEFAULT 'video',
    content TEXT,
    video_url TEXT,
    duration_minutes INT DEFAULT 0,
    sort_order INT DEFAULT 0,
    is_preview BOOLEAN DEFAULT FALSE,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);
```

Also add columns to `learning.courses`:
- `subtitle VARCHAR(500)`
- `short_description TEXT`
- `language VARCHAR(50) DEFAULT 'English'`
- `learning_outcomes TEXT[] DEFAULT '{}'`
- `requirements TEXT[] DEFAULT '{}'`
- `target_audience TEXT[] DEFAULT '{}'`
- `promo_video_url TEXT`
- `gallery TEXT[] DEFAULT '{}'`
- `discount_price NUMERIC(10,2)`
- `enrollment_limit INT DEFAULT 0`
- `visibility VARCHAR(20) DEFAULT 'public'`
- `status VARCHAR(20) DEFAULT 'draft'`

## Existing Infrastructure Reused

- **File upload**: `routes/upload.js` (POST /api/upload, POST /api/upload/video) + `js/shared/image-upload.js` drop-zone widget
- **Auth**: `routes/middleware.js` `requireInstructor` middleware
- **Styles**: `styles/instructor-dashboard.css` patterns (gold accent, sidebar layout)
- **API client**: `js/shared/api.js` for fetch calls

## What's NOT in Scope (for now)

- Drag-and-drop reordering (use up/down buttons instead — simpler, no library needed)
- Real-time video transcoding
- Course versioning
- Bulk lesson import
- AI course outline generation
